# Text Editor

A modern legal document editor built with Next.js, TipTap, and Supabase.

Designed to provide a clean writing experience with rich text editing, autosaving, document management, and a polished interface.

---

## Features

- Rich text editor
- Document autosave
- Multi-document management
- Image uploads
- Dark/light mode
- Modern editor UI
- Supabase document persistence
- Responsive layout

---

## Tech Stack

### Frontend
- Next.js 14
- TypeScript
- Tailwind CSS
- TipTap Editor

### Backend
- Supabase
